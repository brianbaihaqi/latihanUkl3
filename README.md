Program ini berfungsi untuk menghitung faktorial dari suatu bilangan bulat positif yang dimasukkan oleh pengguna. Menggunakan kelas Scanner, program membaca input angka dari keyboard. Jika angka yang dimasukkan kurang dari nol, program akan menampilkan pesan bahwa bilangan harus positif. Jika valid, program menggunakan perulangan for untuk mengalikan angka dari 1 hingga nilai yang dimasukkan, dan hasilnya disimpan dalam variabel faktorial. Setelah perhitungan selesai, hasil faktorial ditampilkan ke layar.
